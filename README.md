# Tokeny
| Nazwa      | Opis               | Wartość   | Kod |
|------------|--------------------| --------- | --- |
| NUMBER     | Liczba całkowita   | [0-9]+    | 1   |
| IDENTIFIER | Zmienna            | [a-zA-z]+ | 2   |
| PLUS       | Operator +         | +         | 3   |
| MINUS      | Operator -         | -         | 4   |
| MULTIPLY   | Operator *         | *         | 5   |
| DIVIDE     | Operator /         | /         | 6   |
| LPAREN     | Nawias otwierający | (         | 7   |
| RPAREN     | Nawias zamykający  | )         | 8   |
